HTML Browser Help

This is an HTML browser, not a web browser. Links starting with "http" will be opened in an external web browser, all others in this view. The tooltip on any link will display you its target.

To close a "Help" window, press the ESCAPE key, or click top right corner.

Font Magnifiction

You can magnify or reduce the text font at any time using "Ctr +" or "Ctrl -" or the context menu on the text-area. Once this was done, all views opened afterwards will show the same font sizes.

Chapters Overview

The "Chapters:" overview in top toolbar lists all headings and elements with id in the displayed documentation. Opening the choice and clicking an item will jump to the selected chapter. This is a regular navigation, so "Back" and "Forward" will work as expected afterwards.

Navigation

There are four navigation buttons:

  1. "Back" will go back to the previously viewed document
  2. "Up" will jump to the start-point of your navigation
  3. "Forward" will will go to the document from where you went "Back"
  4. "Reload" will refresh the current documentation from storage
