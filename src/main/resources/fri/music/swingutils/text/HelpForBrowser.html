HTML Browser Help

This is an HTML-browser for viewing the documentations of this application.

You can magnify or reduce the text font at any time using "Ctr +" or "Ctrl -" or the context menu on the text-area. Once this was done, all views opened afterwards will show the same font sizes.

HTTP-links will be opened in an external web-browser, all others in this view.

Navigation

There are four navigation buttons:

  1. "Back" will go back to previous location
  2. "Top" will jump back to the start point of any navigation path
  3. "Forward" will work after "Back" and will go forward to previous location
  4. "Reload" will refresh the current documentation from its storage

Chapters

The chapters overview lists all headings in the displayed documentation. Opening the choice and clicking an item will jump to the selected chapter. This is a regular navigation, so "Back" and "Forward" will then work as expected.

Close

To close this window, press ESCAPE key, or click top right corner.
