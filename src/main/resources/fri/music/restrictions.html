Restrictions of this Sofware

This project is educational. Its intention was not production of anything other than knowledge.

This project is restricted to 12-tone scales.

You can not try out your own 12-tone scale except when you implement it and recompile the project, currently there is no tool for building scales dynamically at runtime. If you want to implement such, then implement interface ToneSystem or extend AbstractToneSystem. For just-intonations, you can add new Intervals and ChromaticScales to class JustIntonation.
