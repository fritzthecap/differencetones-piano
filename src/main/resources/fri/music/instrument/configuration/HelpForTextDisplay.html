This documentation refers to Java class TuningsAndPurityCheckLauncher.

Text Displays of Tunings, Difference Tones and Purity Checks

All three tab-panels will generate text output according to the settings you can do. Each text window can be closed by pressing ESCAPE when focused, and they all will close automatically when the originator-window gets closed.

Tunings Display

A compound configuration of 12-tone scales provides access to text output of IPN-names, frequencies, cents and more. Please see the "Help" button there on bottom right.

Difference-Tones

In the top part of the panel you can configure the tuning that you want to use for calculation of difference-tones and intervals generating them.

In the middle part you can calculate difference-tones from the interval you can enter in the two text-fields. Use CURSOR-UP and -DOWN to scroll to adjacent tones. The checkbox "Display Only Primary" will control whether you see only the primary or also the secondary and tertiary difference-tone in the text output window.

In the bottom part you can enter a difference-tone in the text-field and display all intervals that would generate it, according to the tuning settings on top. Use the two choices for narrowest and widest interval to limit the findings. Push the button "Display Intervals" to display intervals that would generate the given tone as difference-tone.

The "For a Tone Range" button on bottom opens another configuration dialog where you can enter the lowest tone and number of octaves for a whole tone-range you want to display intervals for. Push the "Display Intervals for All" button to open a window that shows a table where

  • every row is a semi-tone within the given range, and
  • every column shows an interval that is able to generate that tone as difference-tone.
  • Every table-cell either contains the tone-pair that will generate that tone or will be empty when it won't.

You can change back to the "Text Displays" dialog and change the tuning and deviation tolerance there. Doing so, you don't need to restart the intermediate "Tone Range" dialog, you can just push the "Display Intervals for All" button once again to display a new result-table for the altered parameters.

Purity Check

This is for JustIntonation only. The configuration for the check report lets customize the insights into "unjust" intervals and triads of these tunings. Please see the "Help" button there on bottom right.
