Text Displays of Tunings, Difference Tones and Purity Checks

All three tab-panels will generate text output according to the settings you can do. Each text window can be closed by pressing ESCAPE when focused, and they all will close automatically when the originator-window gets closed.

Tunings Display

A compound configuration of 12-tone scales provides access to text output of IPN-names, frequencies, cents and more. Please see the "Help" button there on bottom right.

Difference-Tones

In the top part of the panel you can configure the tuning that you want to use for calculation of difference-tones and intervals generating them.

In the middle part you can calculate difference-tones from the interval you can enter in the two text-fields. Use CURSOR-UP and -DOWN to scroll to adjacent tones. The checkbox "Display Only Primary" will contro whether you see only the primary or also the secondary and tertiary difference-tone in the text output.

In the bottom part ypu can enter a difference-tone in the text-field and display all intervals that would generate it, according to the tuning settings on top. Use the two choices for narrowest and widest interval to limit the findings.

Purity Check

This is for JustIntonation only. The configuration for the check report lets customize the insights into "unjust" intervals and triads of these tunings. Please see the "Help" button there on bottom right.
